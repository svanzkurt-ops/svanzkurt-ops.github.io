# svanzkurt-ops.github.io
Welcome to my professional portfolio! This is a fully responsive, modern portfolio website showcasing my skills, projects, achievements, and activities.

## 📋 Features

### ✨ Sections Included
- **Home/Hero** - Eye-catching introduction with call-to-action buttons
- **About Me** - Personal background and statistics
- **Skills** - Technical skills with progress bars (Frontend, Backend, Tools)
- **Achievements** - Recent accomplishments and recognition
- **Projects** - Portfolio of 6+ recent projects with descriptions
- **Activities** - School and outside activities timeline
- **Contact** - Contact information and message form
- **Footer** - Social media links and copyright

### 🎨 Design Features
- **Responsive Design** - Works perfectly on desktop, tablet, and mobile devices
- **Modern UI** - Clean, professional, and visually appealing
- **Smooth Animations** - Scroll animations, hover effects, and transitions
- **Interactive Elements** - Active navigation highlighting, form validation
- **Accessibility** - Semantic HTML, keyboard navigation support
- **Performance Optimized** - Fast loading with lazy loading support

### 🛠️ Technologies Used
- **HTML5** - Semantic markup
- **CSS3** - Advanced styling with gradients, animations, and flexbox/grid
- **JavaScript (Vanilla)** - No dependencies, pure JavaScript for interactivity
- **Font Awesome Icons** - Beautiful icon library
